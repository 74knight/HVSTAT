Delete Database

If you are facing a problem with the database, and the problem cannot be resolved, delete the database.

In order to re-create the database, reload the page. After that, you can import data previously exported.

Monster Database Migration

Migrate to New Database

HVSTAT will now use the new IndexedDB database instead of the localStorage database. You can migrate your old database into the new database.

If you have already scanned, note that this operation will overwrite existing data on the new database.

Delete Old Monster Database (localStorage)

Currently your old monster database occupies MB on the localStorage. In order to free up space for other HV scripts, delete the old database after migration.
